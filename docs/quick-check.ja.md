# CORE設計フレームワーク クイックチェックシート

## 🔍 使い方

このチェックシートは、デザインレビューや自己評価の際に素早く使える簡易版です。 各質問に「はい/いいえ」で回答し、「いいえ」の項目を重点的に検討しましょう。
