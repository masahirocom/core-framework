# CORE フレームワーク（日本語）

## 概要
**COREフレームワーク**は、「問い」によって設計やアイデアを点検・洗練させるためのシンプルで強力な設計支援フレームワークです。  
プログラマー、ノンプログラマーの両方が使えるように設計されています。

## 特徴
- たった4つの「問い」で構成
- 抜け・矛盾・曖昧な前提に気づける
- ソフトウェア設計、ビジネス設計、教育、研究など幅広く活用可能
- 図解、テンプレート、活用例付き
